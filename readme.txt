== Description ==

***Mine EduCN*** is a powerful, lightweight, and modern free education WordPress theme designed for solo instructors, coaches, and educational institutions to create and sell online courses with unparalleled speed and ease.

Mine EduCN is seamlessly integrated [with Mine Cloudvod](https://wordpress.org/plugins/mine-cloudvod/). It optimizes the templates for the plugin's course listing page, single course page, course schedule, course favorites page, course order page, and more. Additionally, it provides essential block patterns to help you present your content beautifully.
